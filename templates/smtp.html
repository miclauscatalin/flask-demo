Setup Instructions

1. Get Brevo SMTP Credentials
• Login to Brevo Dashboard
• Go to Account → SMTP & API
• In "Your SMTP Settings" find:
  - SMTP Server
  - Port (usually 587)
  - Login (SMTP username)
  - Generate an SMTP key for password
2. SMTP Credentials Summary
For Brevo:
Host: smtp-relay.brevo.com
Port: 587
Username: Your Brevo login email
Password: SMTP key (from SMTP settings)
